Project1 Readme file

-----------------------------------------------------------------------
This project was created based on R language and written using R Studio.
The packages needed for this project are:
1. MASS
2. ISLR
3. Metrics
4. lars
5. caret
6. mlbench
7. randomForest
8. nnet
9. segmented
-------------------------------------------------------------------------
Running Instructions
1. Install all the packages needed in R Studio
2. Open the project file.(Network_backup.R refers to problems 1, 2 and 3, Boston_housing.R refers to problems 4 and 5)
3. Change the address for loading data.
3. Compile and run the projects
